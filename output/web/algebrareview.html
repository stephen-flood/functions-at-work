    Appendix A Algebra Review

    Section A.1 Laws of Algebra

    Distributing and FOIL-ing.

    Distribute \(a\) to all terms:
    \begin{equation*} a \cdot (b+c) = a\cdot b + a \cdot c \end{equation*}
    The best way to remember this is by drawing arcs from \(a\) to \(b\) and \(c\)
    FOIL the expression:
    \begin{equation*} (a + b) \cdot (c + d) = (a\cdot c) + (a \cdot d) + (b\cdot c) + (b\cdot d) \end{equation*}
    The best way to remember this formula is by drawing arcs from \(a\) and \(b\) to \(c\) and \(d\text{.}\)

    Example A.1. Understanding FOIL-ing.

    The mathematical expression for simplifying \((a+b)\cdot(c+d)\) can look a bit mysterious. To understand why it looks the way it does, it is helpful to remember the connection between multiplication and area.
    For example, if you have a rectangle of side lengths \(5\) and \(2\text{,}\) then the rectangle has area \(5\cdot 2 = 10\text{.}\) Visually, we have the following picture.
    Suppose instead we increase the lengths of the sides to \(8=5+3\) and \(6 = 2+4\text{.}\) Let’s see how this changes our area.
    The area given by \((5+3)\cdot(2+4)\) has four regions, each with their own area
    \begin{equation*} 5\cdot 2 + 5\cdot 4 + 3 \cdot 2 + 3\cdot 4 \end{equation*}
    The same mathematical pattern occurs when you have variables in your side lenghts. Suppose you have a rectangle with one side \(3x + 2\) and another side \(x+5\text{.}\) Then the area of the rectangle still has four regions
    Adding up the areas of those regions gives us the same thing as FOIL-ing
    \begin{equation*} (3x+2)(x+5) = (3x\cdot x) + (3x\cdot 5) + (2\cdot x) + (2\cdot 5) \end{equation*}
    Simplifying that down we get
    \begin{equation*} (3x+2)(x+5) = 3x^2 + 17x + 10 \end{equation*}
    Of course, with practice, it is faster and easier to learn to use the formula for FOIL-ing than would be to go through the area derivation each time.

    Exploration A.1.

    Use the laws of algebra listed above to simplify the following expressions completely.

    (a)

    \(5(x^2 - 3x + 2)\)
    Solution.
    We must distribute the 5 to every term of the expression
    \begin{align*} 5(x^2 - 3x + 2) \amp = 5x^2 - 15 x + 10 \end{align*}

    (b)

    \((-1)(x - 5)\)
    Solution.
    We must distribute a \(-1\) to each term
    \begin{align*} (-1)(x - 5) \amp = (-1)x - (-1)5 \\ \amp = -x + 5 \end{align*}

    (c)

    \((x+6)(x-2)\)
    Solution.
    We must foil the expression using the formula
    \begin{equation*} (a + b) \cdot (c + d) = (a\cdot c) + (a \cdot d) + (b\cdot c) + (b\cdot d) \end{equation*}
    with \(a=x\text{,}\) \(b=6\text{,}\) \(c=x\text{,}\) and \(d=-2\text{.}\) In other words, we have
    \begin{align*} (x+6)(x-2)\amp = (x\cdot x) + (x\cdot -2)+ (6\cdot x) + (6\cdot -2)\\ \amp = x^2 - 2x + 6x - 12 \amp \\ \amp = x^2 + 4x - 12 \end{align*}

    (d)

    \((5x-1)(2x+1)\)
    Solution.
    We must foil the expression using the formula
    \begin{equation*} (a + b) \cdot (c + d) = (a\cdot c) + (a \cdot d) + (b\cdot c) + (b\cdot d) \end{equation*}
    with \(a=5x\text{,}\) \(b=-1\text{,}\) \(c=2x\text{,}\) and \(d=1\text{.}\) In other words, we have
    \begin{align*} (5x-1)(2x+1)\amp = (5x\cdot 2x) + (5x\cdot 1)+ (-1\cdot 2x) + (-1\cdot 1)\\ \amp = 10x^2 + 5x - 2x - 1 \amp \\ \amp = 10x^2 + 3x - 1 \end{align*}

    (e)

    \((2x+3)^2\)
    Solution.
    Squaring an expression means multipling it by itself twice. In other words,
    \begin{align*} (2x+3)^2 \amp = (2x+3)\cdot (2x+3)\\ \amp = (2x\cdot 2x) + (2x\cdot 3) + (3\cdot 2x) + (3\cdot 3)\\ \amp = 4x^2 + 6x + 6x + 9\\ \amp = 4x^2 + 12x + 9 \end{align*}

    (f)

    \((-3)(2x+3)^2\)
    Solution.
    We must use order of operations to decide which operation to perform first. Remember that PEMDAS (parentheses, exponents, multiplcation/division, addition/subtraction) tells us which operations attach more "strongly", and must be performed first.
    1. Here, the parentheses group the expression into two parts: \(A\cdot B^2\) with \(A=(-3)\) and \(B=(2x+3)\text{.}\)
    2. Exponentiation is stronger than multiplication, so we must first square \((2x+3)\text{.}\) As in the previous problem, we have
      \begin{align*} (-3)(2x+3)^2\amp = (-3)\cdot (4x^2 + 12x + 9) \end{align*}
    3. Now, we can distribute \(-3\) to each term.
      \begin{align*} \amp = (-3)\cdot (4x^2 + 12x + 9) \\ \amp = (-3\cdot 4) x^2+ (-3\cdot 12)x + (-3\cdot 9)\\ \amp = -12x^2 - 36x - 27 \end{align*}

    (g)

    Factor the following expression as much as possible \(9x - 15\)
    Solution.
    First, factor each term in the expression. There is a 3 in each term, so we can factor it out.
    \begin{align*} 9x - 15 \amp = (3\cdot 3)x - (3\cdot 5) \\ \amp = 3 (3x - 5) \end{align*}
    There are no remaining common terms, so we are done factoring.

    (h)

    Factor the following expression as much as possible \(x^2 + x - 6 \)
    Solution.
    We want to write \(x^2 + x - 6 \) as an expression of the form \((a x + b) (c x + d) \)
    \begin{equation*} (a x + b) (c x + d) = (ab)x^2 + (ad+cb)x + bd \end{equation*}
    We want to pick \(a,b,c,d\) to get the right hand side to equal \(x^2+x-6\text{.}\) There are three terms to consider
    • We need \((ab)x^2=x^2\text{,}\) so set \(a=b=1\text{.}\) Plugging that into our factored form, we get
      \begin{equation*} (x + b)(x+d) = x^2 + (d+b)x + bd \end{equation*}
    • We need the third term to be \(db=-6\text{.}\) Factoring gives \(db = (-1)\cdot 2\cdot 3\text{.}\) We must divide the factors \(-1\text{,}\) \(2\text{,}\) \(3\) between \(b,d\text{.}\)
    • To pick exact values for \(b,d\text{,}\) recall that we need \(b+d=1\text{.}\) Setting \(b=3\) and \(d=(-1)2\text{,}\) we can confirm
      \begin{align*} bd \amp = (3)(-1)(2) = -6\\ b+d \amp = 3 + (-1)(2) = 1 \end{align*}
      as desired.
    Putting it together, we get
    \begin{align*} x^2 + x - 6 \amp = (x+3)(x-2) \end{align*}

    (i)

    Factor the following expression as much as possible \(2x^7 + 5x^3 \)
    Solution.
    First, factor each term in the expression. There are three occurances of \(x\) in each term, so we can factor them out.
    \begin{align*} 2x^7 + 5x^3 \amp = (2\cdot x \cdot x\cdot x\cdot x\cdot x\cdot x\cdot x) + (5\cdot x\cdot x\cdot x) \\ \amp = x\cdot x\cdot x (2\cdot x\cdot x\cdot x\cdot x + 5)\\ \amp = x^3 (2 x^4 + 5) \end{align*}
    There are no remaining common terms, so we are done factoring.

    Axiom A.2. Laws of Fractions.

    Suppose that \(b,e\neq 0\)
    • \(\displaystyle \frac{a}{b} + \frac{c}{d} = \frac{ad + cb}{bd}\)
    • \(\displaystyle \frac{a}{b} + \frac{c}{b}= \frac{a+c}{b}\)
    • \(\displaystyle \frac{a}{b} \cdot \frac{c}{e} = \frac{a\cdot c}{b\cdot e}\)
    • \(\displaystyle \frac{a\cdot e}{b\cdot e} = \frac{a}{b}\)
    • \(\displaystyle \frac{a}{b} - \frac{c}{b} = \frac{a-c}{b}\)
    • \(\displaystyle \frac{1}{\ \frac{b}{e}\ }= \frac{e}{b}\)
    • \(\displaystyle a\cdot \frac{c}{b}= \frac{a\cdot c}{b}\)

    Exploration A.2.

    Use the laws of algebra listed above to simplify the following expressions completely.

    (a)

    \(\dfrac{ab - a}{b-1}\)
    Solution.
    \begin{equation*} \dfrac{ab - a}{b-1} = \dfrac{a(b-1)}{1\cdot (b-1)} = \dfrac{a}{1} \end{equation*}

    (b)

    Simplify \(\dfrac{a-b}{b-a}\)
    Solution.
    \begin{align*} \dfrac{a-b}{b-a} \amp = \dfrac{a-b}{-a + b }\\ \amp = \dfrac{a-b}{(-1) ( a - b)}\\ \amp = \dfrac{1}{-1}\\ \amp = -1 \end{align*}

    Axiom A.3. Laws of Exponents.

    Let \(a,b> 0\) in the following.
    • \(\displaystyle \sqrt[r]{a}= a^{1/r}\)
    • \(\displaystyle a^{r+s} = a^r\cdot a^s \)
    • \(\displaystyle {\left(a^{r}\right)}^s= a^{r\cdot s}\)
    • \(\displaystyle {\left(a\cdot b\right)}^{r}= a^{r}\cdot b^{r}\)
    • \(\displaystyle \sqrt[r]{a\cdot b}= \sqrt[r]{a}\cdot \sqrt[r]{b}\)
    • \(\displaystyle a^{r-s}= \frac{a^r}{a^s}\)
    • \(\displaystyle a^{-s}= \frac{1}{a^s}\)
    • \(\displaystyle a^{s}= \frac{1}{a^{-s}}\)
    • \(\displaystyle \frac{1}{a^{s}} = \left(\frac{1}{a}\right)^s\)

    Example A.4. Fractions which do NOT simplify easily.

    Doesn’t simplify at all: \(\dfrac{3}{x+1}\)
    Must find a common denomiator before you can combine terms:
    \begin{equation*} \dfrac{x+1}{x-1} + \dfrac{1}{x} = \dfrac{\,x\cdot (x+1) + (x-1)\cdot 1\,}{x\cdot (x-1)} = \dfrac{x^2+2x-1}{x^2-x} \end{equation*}

    Example A.5. Exponents and Products that do NOT simplify easily.

    Doesn’t simplify at all: \(\sqrt{x+1} = (x+1)^{1/2}\)
    You must FOIL to simplify: \((x+1)^2 = (x+1)(x+1) = x^2 + 2x + 1\)

    Section A.2 Intervals and Graphs

    Subsection A.2.1 Interval Notation

    When using interval notation to represent sets, square brackets include the endpoint and parentheses exclude the endpoint. Multiple intervals can be combined using \(\cup\) (or U).
    • \([-1,1)\) represents the numbers \(x\) such that \(-1\leq x <1\text{.}\)
    • \([0,\infty)\) represents the numbers \(x\) such that \(0\leq x \text{.}\)
    • \((-\infty,-1)\bigcup (1,2]\) represents the numbers \(x\) such that either \(x<-1\) or \(1<x\leq 2\text{.}\)
